# Excel Project Documentation

This repository contains an Excel project where various data analysis and preparation techniques were used to create an interactive dashboard.

## Techniques Used

1. **XLOOKUP**
   - [Microsoft Documentation on XLOOKUP](https://support.microsoft.com/en-us/office/xlookup-function-5d8e1c15-1e3d-4d8b-bb5e-bb5eddc18b62)
   - This function was used to search for and return a value in a specified range by matching it with a value in another range.

2. **INDEX MATCH**
   - [Microsoft Documentation on INDEX](https://support.microsoft.com/en-us/office/index-function-a5dcf0dd-996d-40a4-a822-b56b061328bd)
   - [Microsoft Documentation on MATCH](https://support.microsoft.com/en-us/office/match-function-e8dffd45-c762-47d6-bf89-533f4a37673a)
   - Combined, these functions were used to look up a value in a table and return a corresponding value from another column.

3. **Multiplication Formula for Sales**
   - [Microsoft Documentation on Multiplication](https://support.microsoft.com/en-us/office/multiply-and-divide-numbers-in-excel-d7cad028-dba1-4052-8eaf-8808f63d7b7f)
   - This formula was used to calculate total sales by multiplying quantity sold by price per unit.

4. **Multiple IF Functions**
   - [Microsoft Documentation on IF](https://support.microsoft.com/en-us/office/if-function-69aed7c9-4e8a-4755-a9bc-aa8bbff73be2)
   - Nested IF functions were used to evaluate multiple conditions and return different results based on the outcomes.

5. **Date Formatting**
   - [Microsoft Documentation on Date Formatting](https://support.microsoft.com/en-us/office/format-a-date-the-way-you-want-8e10019e-d5d7-4e4a-9307-0e49d2cda589)
   - This was used to ensure dates were displayed in a consistent format across the project.

6. **Number Formatting**
   - [Microsoft Documentation on Number Formatting](https://support.microsoft.com/en-us/office/available-number-formats-in-excel-0afe8f52-97db-41f1-b972-4b46e9f1e8d2)
   - Applied to ensure numerical data was easy to read and interpret.

7. **Check For Duplicates**
   - [Microsoft Documentation on Remove Duplicates](https://support.microsoft.com/en-us/office/find-and-remove-duplicates-00e35bea-b46a-4d5d-b28e-66a552dc138d)
   - Used to identify and remove duplicate entries in the dataset.

8. **Convert Range to Table**
   - [Microsoft Documentation on Convert Range to Table](https://support.microsoft.com/en-us/office/create-and-format-tables-660cbe2e-2fca-4c49-b17a-48d8a53dbe7a)
   - This was done to leverage table features such as filtering, sorting, and structured references.

9. **Pivot Tables and Pivot Charts + Formatting**
   - [Microsoft Documentation on Pivot Tables](https://support.microsoft.com/en-us/office/create-a-pivottable-to-analyze-worksheet-data-a9a84538-bfe9-40a9-a8e9-f99134456576)
   - [Microsoft Documentation on Pivot Charts](https://support.microsoft.com/en-us/office/create-a-pivotchart-fc60c07b-a013-41a5-9292-e9b5ae8aaeff)
   - Used to summarize and visualize data effectively.

10. **Insert Timeline + Formatting**
    - [Microsoft Documentation on Insert Timeline](https://support.microsoft.com/en-us/office/use-a-timeline-to-filter-dates-efbb25e7-c80e-40a9-826d-3c92cb177f8c)
    - Used to filter data based on time periods in PivotTables.

11. **Insert Slicers + Formatting**
    - [Microsoft Documentation on Insert Slicers](https://support.microsoft.com/en-us/office/use-slicers-to-filter-data-249f966b-a9d5-4b0f-b31a-12651785d29d)
    - Used to add visual filters to PivotTables and PivotCharts for better data analysis.

## Project Structure

- `data/`: Contains the raw data files.
- `analysis/`: Contains the Excel files with the analysis and dashboard.
- `documentation/`: Contains additional documentation and resources.

## Getting Started

To get started with this project, clone the repository and open the Excel files in the `analysis/` folder. Follow the steps documented above to understand the techniques used.
